# LARAVEL CMS 
## Support

* Laravel: 5.4 & 5.5 COMING SOON
* PHP: 7.0

## Commands 

* Run all migrations (IMPORTANT)

    php artisan migrate

* To publish config files

  php artisan vendor:publish --tag=config

* To publish assets

  php artisan vendor:publish --tag=public --force
